ANNOYING_AUTOMOBILE_DRIVER_SOUND_TRACK_LOOPER

Use this web application to generate obnoxious noises which play continuously for indefinitely many loops.

Multiple sound tracks can be played simultaneously by clicking the "add another sound file menu" button.

For best results, select at least the first two sound tracks in the sound file menu. (Those first two sound tracks feature exclusively car horn beeps and engine revving noises interspersed with silence).

To increase the randomness of the sound effects, open multiple browser tabs with this application and click the "start sound track looper" buttons at different times.

1) Select one or multiple sound files from the list below.

2) Click the start_sound_track_looper() button to start playing the selected sound file(s).

This section will be updated with information about selected sound tracks after the start button is clicked.
